# Advanced Settings Reference

This document provides a reference to all advanced settings configuration categories for EAA applications. The `advanced_settings` parameter accepts a JSON-encoded string containing various configuration options.

For detailed application type-specific configurations, restrictions, and example files, see [Application Type Configurations](./app-type-configurations.md).

## Advanced Settings Categories

The following documents provide detailed parameter documentation for each category:

- [Health Check Parameters](./healthcheck_advsettings.md) - Configure health checks for application servers
- [Server Load Balancing Parameters](./server_load_balancing_advsettings.md) - Configure load balancing and session stickiness
- [Enterprise Connectivity Parameters](./enterprise_connectivity_advsettings.md) - Configure connection pooling and timeouts
- [Tunnel Client Parameters](./tunnel_client_advsettings.md) - Configure tunnel-specific behavior
- [SSL and WebSocket Configuration Parameters](./ssl_websocket_advsettings.md) - Configure SSL verification, WebSocket, and connection settings
- [Authentication Parameters](./authentication_advsettings.md) - Configure user-facing and application-to-origin authentication
- [CORS Parameters](./cors_advsettings.md) - Configure Cross-Origin Resource Sharing
- [TLS Suite Parameters](./tls_suite_advsettings.md) - Configure TLS cipher suites
- [Miscellaneous Parameters](./miscellaneous_advsettings.md) - Configure various application behavior settings
- [RDP Configuration Parameters](./rdp_advsettings.md) - Configure Remote Desktop Protocol settings
- [Basic Configuration Parameters](./basic_configuration_advsettings.md) - Configure core application settings
